Two days, barely slept, but it runs: a **match-3** for the @blupixapp universe.

Quick note before the screenshots — none of the mechanics are random. They follow
the collection's three themes: **reveal, merge, evolve.**

**[ 📸 Start screen ]**

**Reveal.** Matches fill a meter; top it off and a reroll charge hatches — the
button lights up, you spend it, the board reshuffles. Each run also rolls a
**vibe** that bends the rules (extra moves, score multipliers, evolution auras).

**[ 📸 Reroll lit up + vibe intro ]**

**Merge.** When a color levels up you pick a **fusion partner** — Red × Yellow →
Orange, and so on. That locks which family you're building toward.

**[ 📸 Fusion modal ]**

**Evolve.** Matches feed essence; colors climb **T1 → T2 → T3 → T4.** Hit T4 and
you've got an apex form (Pyronix, Cryonix, Umbryx…). But evolving one color decays
the others — you can't greed all 8, you commit to one pet. Win and it lands in your
gallery.

**[ 📸 Mid-run board + victory card ]**

One thing I'm quietly proud of: the leaderboard is **real.** Every win is replayed
server-side before it counts — no fake scores, no cherry-picked seeds.

**[ 📸 Leaderboard ]**

That's it. Two sleepless days, a working game, and a clear next step: wiring it to
your onchain pets via **BluKit.** Go reveal something. 🟦

Два дня, почти без сна, но оно работает: **match-3** для вселенной @blupixapp.

Сразу оговорюсь — механики тут не случайны. Они повторяют три темы коллекции:
**reveal, merge, evolve.**

**[ 📸 Стартовый экран ]**

**Reveal.** Матчи копят шкалу; добил — вылупляется заряд reroll, кнопка
загорается, тратишь — поле пересобирается. Ещё каждый забег рулит **вайб**,
который гнёт правила (доп. ходы, множители счёта, ауры эволюции).

**[ 📸 Reroll загорелся + интро вайба ]**

**Merge.** Когда цвет апается, выбираешь **партнёра для фьюжна** — Red × Yellow →
Orange и т.д. Это фиксит, какую семью ты строишь.

**[ 📸 Модалка фьюжна ]**

**Evolve.** Матчи кормят эссенцию; цвета лезут по тирам **T1 → T2 → T3 → T4.**
Дошёл до T4 — апекс-форма (Pyronix, Cryonix, Umbryx…). Но эволюция одного цвета
роняет остальные — зажать все 8 нельзя, коммитишься в одного пета. Выиграл — форма
падает в галерею.

**[ 📸 Поле в забеге + карточка победы ]**

Чем тихо горжусь: лидерборд тут **настоящий.** Каждая победа переигрывается на
сервере перед зачётом — без фейковых счётов и выцепленных сидов.

**[ 📸 Лидерборд ]**

Вот и всё. Два дня без сна, рабочая игра и понятный следующий шаг — связать её с
твоими ончейн-петами через **BluKit.** Иди ревильни что-нибудь. 🟦
